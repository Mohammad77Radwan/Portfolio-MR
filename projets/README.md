# Projets

Ce dossier contient les réalisations techniques présentées au jury et aux recruteurs.

## Organisation recommandée
- Un dossier par projet.
- Chaque projet contient un `README.md` détaillé.
- Les captures sont stockées dans un sous-dossier `medias/` du projet.

## Exemple de structure
- `projet-gestion-stock/`
- `projet-app-mobile/`
- `projet-web-fullstack/`
