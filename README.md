## Démarrage rapide
```bash
cd /workspaces/Portfolio-MR/portfolio-next
npm install
npm run dev
```

Ouvrir ensuite [http://localhost:3000](http://localhost:3000).

## Structure
- `portfolio-next/` : application Next.js moderne
- `documentation/` : exigences, oral, évaluation et conformité
- `projets/` : modèles et comptes rendus projets
- `stages/` : pièces de stage

## Remarque
Les documents E4/E5 importants sont centralisés dans `documentation/` et les preuves projet dans `projets/`.
