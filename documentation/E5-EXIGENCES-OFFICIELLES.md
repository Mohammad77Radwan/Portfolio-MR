# E5 - Exigences officielles (synthese operative)

Ce document transforme les consignes enseignant en actions concretes.

## 1) Documents a fournir et partager
- Tableau de competences / tableau de synthese
- Lien du portfolio GitHub fonctionnel
- Attestations de stage 1ere et 2eme annee signees
- Diaporama de presentation

## 2) Verifications obligatoires
- L'enseignant a bien acces aux documents partages
- Le portfolio est accessible en format electronique
- Le README principal explique clairement le contenu
- Navigation et compatibilite testees avant l'epreuve

## 3) Deroulement oral
- Partie 1: presentation du parcours (10 minutes max)
- Partie 2: echange avec le jury (30 minutes max)

## 4) Attendus du jury
- Capacite a justifier les choix techniques
- Maitrise des realisations et du code
- Mise en relation claire: competence -> preuve -> resultat
- Communication orale professionnelle

## 5) Risques de penalites
- Portfolio inaccessible: -10 points
- Absence tableau de synthese: -2 points

## 6) Vigilance conformite
- Couverture complete des competences du bloc
- Environnement technologique conforme au referentiel
- Pieces presentes, datees et exploitables
