# Checklist Conformite E4/E5 (CCF)

## Dossier obligatoire
- [ ] Portfolio accessible en ligne
- [ ] Portfolio partage sur Google Drive avec l'enseignant
- [ ] Tableau de synthese present
- [ ] Attestation de stage 1ere annee signee
- [ ] Attestation de stage 2eme annee signee
- [ ] Diaporama de presentation pret

## Controle accessibilite (a faire avant l'epreuve)
- [ ] Tous les liens GitHub fonctionnent
- [ ] Lien portfolio teste depuis un navigateur externe
- [ ] Autorisations de lecture Google Drive verifiees
- [ ] Navigation claire sur mobile et desktop
- [ ] README principal explicite le contenu du portfolio
- [ ] Tous les README projets sont completes
- [ ] Documents ouverts sans erreur (PDF, images, liens)

## Preparation orale
- [ ] Script oral 10 min repete au moins 3 fois
- [ ] Jeu de 30 questions techniques revise
- [ ] Temps respecte: 10 min presentation + 30 min echange

## Couverture des competences (Bloc 1)
- [ ] Gerer le patrimoine informatique
- [ ] Repondre aux incidents et demandes
- [ ] Developper la presence en ligne
- [ ] Travailler en mode projet
- [ ] Mettre a disposition un service
- [ ] Organiser son developpement professionnel

## Vigilance jury / penalites
- [ ] Absence du tableau de synthese = -2 points
- [ ] Portfolio inaccessible = -10 points
- [ ] Environnement technologique conforme au referentiel
